- [CAF_5003 - Replace boilerplate text from a document](CAF_5003)
- [CAF_5004 - Identify presence of boilerplate text in a large document](CAF_5004)
- [CAF_5005 - Remove boilerplate text from a large document](CAF_5005)
- [CAF_5006 - Replace boilerplate text from a large document](CAF_5006)
- [CAF_5009 - KeyContent segregation for emails - original email is primary content](CAF_5009)
- [CAF_5010 - KeyContent segregation for emails - last reply is primary content](CAF_5010)
- [CAF_5011 - KeyContent segregation for emails - primary content only](CAF_5011)
- [CAF_5012 - KeyContent segregation for emails - primary and secondary content only](CAF_5012)
- [CAF_5013 - Signature removal from emails](CAF_5013)
- [CAF_5014 - Signature detection in emails](CAF_5014)
- [CAF_5015 - KeyContent segregation for emails - overlapping content](CAF_5015)
